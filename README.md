##  Project Overview

This project analyzes terrorism-related incidents in India from **2000 to 2020** using data from the **Global Terrorism Database (GTD)**.

The aim is to understand:
- how terrorism trends evolved over time,
- which years were the deadliest,
- who bore the maximum human cost,
- how attack methods changed,
- and how major militant groups contributed to terrorism in **Jammu & Kashmir**.
